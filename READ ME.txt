## Project Description
A web-based login system with user authentication using PHP and MySQL. Users can register, login, and access a secure dashboard.

## Technologies Used

- HTML5

## Steps to Run the Project
1. Install XAMPP/WAMP/LAMP
2. Start Apache and MySQL services
3. Copy project folder to htdocs (for XAMPP)
4. Import database using phpMyAdmin
5. Update database credentials in config/database.php
6. Access the application at: http://localhost/your-project-folder

## Database Import Instructions
1. Open phpMyAdmin
2. Create a new database named 'login_system'
3. Click on 'Import' tab
4. Browse and select the 'database.sql' file
5. Click 'Go' to import
